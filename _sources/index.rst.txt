Welcome to GWADAMA's documentation!
===================================

**GWADAMA** (Gravitational Wave Dataset Management) is a collection of ad-hoc classes and
functions intended to automate gravitational-wave data-related operations:

- Wave generation
- Data input
- Train/Test splitting (smart index tracking for Cross-Validation tasks)
- Injections
- Pre-processing (normalization, resampling, whitening, filtering, ...)
- Export to several formats
